# 🚦 RoadSense AI

> **Transforming Community Reports into Intelligent Road Decisions**

RoadSense AI is an AI-powered community traffic intelligence platform that transforms unstructured road incident reports into verified, real-time traffic intelligence. Using multimodal AI, users can report incidents through **text, voice, images, or videos**, while Google Gemini extracts structured information, generates concise summaries, and helps verify incidents. A Machine Learning model predicts traffic impact, and verified incidents are visualized on a live interactive map.

## 🎯 Problem Statement

Current traffic information is scattered across navigation apps, social media, and messaging platforms. Reports are often delayed, duplicated, or unverified, making it difficult for commuters to make informed travel decisions.

RoadSense AI solves this by converting community-generated reports into structured, AI-verified, and actionable traffic intelligence.

---

# ✨ Key Features

- 🚨 Multimodal Incident Reporting (Text, Voice, Image & Video)
- 🤖 AI-Powered Incident Analysis using Google Gemini
- 📍 Automatic GPS & Manual Map Location Selection
- 🔄 Duplicate Incident Detection
- 📊 ML-Based Traffic Impact Prediction
- 📝 AI-Generated Incident Summaries
- 🗺️ Interactive Live Incident Map
- ☁️ Cloud-Based Data Storage with TiDB Cloud
